## Simple proxy server that saves the session when disconnect

# Installation

* Install nodejs on [official website](https://nodejs.org/en/)

* Enter these commands into the terminal:
* `git clone https://github.com/dest4590/MIOXY.git`
* `cd MIOXY`
* `npm install`
* `node index.js`
