## Простой прокси сервер который сохраняет сессию при выходе

# Установка

* Установите NodeJS на [официальном сайте](https://nodejs.org/en/)

* Введите эти команды в терминал:
* `git clone https://github.com/dest4590/MIOXY.git`
* `cd MIOXY`
* `npm install`
* `node index.js`
